# RISC-V (Sub-Set) Out-of-Order Interpreter

## Getting Started

```bash
# for dev
npm run dev # or `yarn dev`

npm test # or `yarn test`
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
