# RISC-V (Sub-Set) Out-of-Order Interpreter

![](public/demo.png)

## Getting Started

```bash
# for dev
npm run dev # or `yarn dev`
# for test
npm test # or `yarn test`
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. [Docs](docs/README.md).
